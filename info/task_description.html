Lets cow say!

Our cow is young and can only say. There is a cow that can be Tux (wiki/Cowsay). You are given some text and your function should format it in the "cow" text.

Let's examine the rules for that:
  • Cow is always the same, only quote changes.
  • Multiple spaces in a row replaced by one space.
  • Top consists from underscore chars. It starts from single space and end before border column.
  • Each line of quote consists from next parts: quote border(1), space(1), line(1-39), space(1), quote border(1).
  • If line less then 40 characters it will fit in one string. String quoted in <>
  • If line greater or equal 40 chars it should be split by next rules:
    • Max line size is 39 chars. If any spaces in line split by most right space(this space is removed from text) otherwise take first 39 chars.
    • After split align all line to same length by adding spaces to the end.
    • First line borders: /\
    • Middle line borders: ||
    • Last line borders: \/
  • Bottom consist form minus. Has same length as top. 
     _____________________________
    / Dog goes woof               \
    | Cat goes meow               |
    | Bird goes tweet             |
    | and mouse goes squeek       |
    | Cow goes moo                |
    | Def go foo                  |
    \ And solution will go to you /
     -----------------------------
            \   ^__^
             \  (oo)\_______
                (__)\       )\/\
                    ||----w |
                    ||     ||
What does the cow say?

Input: Text as a string.

Output: The result for console as a string.

Hint: Read python docs (2.7, 3.3) about formatting styles (str.format and %). Notice for r before string. It is raw strings they use different rules for interpreting backslash escape sequences.

Example: 

checkio('Checkio rulezz') == r'''
 ________________
< Checkio rulezz >
 ----------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
'''
checkio('A longtextwithonlyonespacetofittwolines.') == r'''
 ________________________________________
/ A                                      \
\ longtextwithonlyonespacetofittwolines. /
 ----------------------------------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
'''

checkio('Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.') == r'''
 _________________________________________
/ Lorem ipsum dolor sit amet, consectetur \
| adipisicing elit, sed do eiusmod tempor |
| incididunt ut labore et dolore magna    |
\ aliqua.                                 /
 -----------------------------------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
'''

How it is used: The original Cowsay are written in the Perl programming language, and as such is easily adaptable to system tasks in Unix, such as telling users their home directories are full, they have new mail, etc. Now you will write your own realisation for the old unix program. This concept can learn you how to prepare and format text for the console output.
