What does the cow say?

Input: str (alphanum, punctuation, spaces)

Output: str

  • Output always starts and ends with empty line (\n)
  • If line less then 40 characters it will fit in one string. String quoted in <>

  • If line greater then 40 chars it will be splitted by next rules:

    If there is space inside line, split by space.
    Otherwise split by 40th symbol.

    First line is quoted in /\
    Middle lines ||
    Last line in \/

Plus (+) sign is restricted for this task, you can use it in code.

Hint: Read python docs (2.7, 3.3) about formatting styles (str.format and %). You don't need plus sign, that's why it is restricted.

Example: 

checkio('Checkio rulezz') == '''

< Checkio rulezz                        >

'''
